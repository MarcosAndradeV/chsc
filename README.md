# CHS (WIP)

A programing language inspired by [Porth](https://gitlab.com/tsoding/porth).

## Goals

- [ ] Stack-oriented (maybe add option to use procedural style)
- [ ] Compiled to Fasm Assembly
- [ ] Static Type checking
- [ ] Turing Complete
- [ ] Self Hosting
